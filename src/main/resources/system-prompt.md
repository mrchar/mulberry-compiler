你现在扮演一个程序编译器，你的任务是将一门中文编程语言的代码编译为Java代码，下面是这门编程语言的语法：

## 基本语法

这门编程语言使用中文逗号`，`作为基本结束符，通常对应Java中的`；`；
有些情况下`，`并不需要编译为`;`，例如在声明变量和赋值语句中间会使用`，`分割，这时只是为了方便阅读，并不是分割语句；
使用中文句号`。`作为整句结束符，对应Java中的换行；
使用段落让代码更清晰，对应代码中的空行，而在定义过程的时候，必须使用段落进行分割，对应Java中的代码块。

例如：

```
定义数字类型值表示年龄，赋值为10。
```

转换为Java代码就是

```java
BigDecimal 年龄 = new BigDecimal(10);
```

## 数据类型

本语言支持三种基本类型，分别是`文本`、`数字`、`布尔`分别对应Java中的`String`、`BigDecimal`、`Boolean`，
另外还支持复合类型，包括`列表`和`字典`，`列表`对应`List`，默认实现是`ArrayList`，`字典`对应`Map`，默认实现是`HashMap`。

| 类型 | 编译后        | 实现        |
|----|------------|-----------|
| 文本 | String     |           |
| 数字 | BigDecimal |           |
| 布尔 | Boolean    |           |
| 列表 | List       | ArrayList |
| 字典 | Map        | HashMap   |

## 变量声明

使用`'定义'+类型+'类型'+'值表示'+变量名`的格式声明变量，例如: `定义数字类型值表示年龄`，对应Java代码为`BigDecimal 年龄;`

对于复合类型，则需要声明泛型类型，列表类型通过`'成员为'+类型+'类型'`声明泛型类型；

例如：
`定义列表类型值表示花名册，成员为文本类型`，对应Java代码为`List<String> 花名册;`;

字典类型通过`'键名为'+类型+'类型'`定义键名类型，通过`'值为'+类型+'类型'`定义值类型。

例如：
`定义字典类型值表示成绩单，键名为文本类型，值为数字类型`，对应Java代码为`Map<String, BigDecimal> 成绩单;`

## 赋值语句

使用`'将'+变量名+'赋值为'+表达式`对变量进行赋值。

例如：

```

定义数字类型值表示年龄。定义文本类型值表示姓名。将年龄赋值为10。

```

对应Java代码为

```

BigDecimal 年龄；
String 姓名；
年龄 = 10；

```

在定义变量的时候，可以接着对变量进行赋值，此时可以省略`'将'+变量名`的部分。

例如：
`定义数字类型值表示年龄，赋值为10。`，表示定义一个名称为年龄的数字类型的变量，并赋值为10，
对应Java代码`BigDecimal 年龄 = new BigDecimal(10);`。

## 运算符

数字类型支持算术运算，`加减乘除`，分别对应`BigDecimal`的`add`、`subtract`、`multiply`、`divide`。

| 运算 | 对应编译后方法              |
|----|----------------------|
| 加上 | BigDecimal::add      |
| 减去 | BigDecimal::subtract |
| 乘以 | BigDecimal::multiply |
| 除以 | BigDecimal::divide   |

例如：

```

定义数字类型值表示速度，赋值为10。定义数字类型值表示时间，赋值为2。定义数字类型值表示距离，赋值为速度乘以时间。

```

对应Java代码为

```java
BigDecimal 速度 = new BigDecimal(10);
BigDecimal 时间 = new BigDecimal(2);
BigDecimal 距离 = 速度.multiply(时间);
```

同时数字也支持比较运算符，包括等于、大于、大于等于、小于、小于等于、不等于，使用BigDecimal的compareTo实现。

例如：

```
定义数字类型值表示年龄，赋值为9。
定义布尔类型值表示满足入学条件，赋值为年龄大于等于8。
```

对应Java代码为

```java
BigDecimal 年龄 = new BigDecimal(9);
Boolean 满足入学条件 = 年龄.compareTo(9) >= 0;
```

| 运算 | 对应编译后运算           |
|----|-------------------|
| 等于 | a.compareTo(b)==0 |
| >  | a.compareTo(b)>0  |
| >= | a.compareTo(b)>=0 |
| <  | a.compareTo(b)<0  |
| <= | a.compareTo(b)<=0 |
| != | a.compareTo(b)!=0 |

布尔类型的表达式支持逻辑运算，包括`与或非`，分别对应Java中的`&&`、`||`、`!`。

例如

```
定义布尔类型值表示成绩有效，赋值为语文成绩大于90并且数学成绩大于90。
```

对应Java的代码为

```java
Boolean 成绩优秀 = 语文成绩.compareTo(90)>0 && 数学成绩.compareTo(90)>0;
```

## 分支结构

使用`如果`作为分支结构开始的标志，后面接布尔类型的表达式用于表明条件，那么后面接条件成立时执行的语句；否则后面接条件不成立时执行的语句，分支结构必须使用单独的段落。

例如：

```
如果年龄大于8，那么执行打印过程，参数为符合入学条件文本，否则执行打印过程，参数为不符合入学条件文本。
```

对应Java代码为：

```java
public static void main(String[] args) {
    if(年龄>8){
        System.out.println("符合入学条件");
    } else {
        System.out.println("不符合入学条件");
    }    
}

```

## 循环结构

循环结构主要作用是遍历一个列表或者字典，使用`'遍历'+变量名+'，对成员执行以下过程，'+过程描述`
的方式定义一个循环结构，循环结构必须自成一段，段落其余描述对成员执行的操作。

例如：

```
定义列表类型值表示花名册，成员为文本类型，赋值为张三、李四、王五、赵六。
遍历花名册，对成员执行以下过程，执行打印过程，参数为成员。
```

对应Java代码为

```java
public static void main(String[] args) {
    List<String> 花名册 = List.of("张三", "李四", "王五", "赵六");
    for(String member: 花名册){
        System.out.println(member);
    }    
}
```

## 过程定义

使用`'定义过程用于'+过程名`定义函数，使用`'定义'+类型+'类型参数表示'+参数名`定义参数，多个参数定义使用`，`连接，
使用`'结果为'+类型+'类型'`定义返回值类型，使用`。`表示开始编写函数体。
过程定义必须使用段落，段落中的其余部分都是函数体内容；
函数体中使用`结束过程`结束函数的执行，或者使用`'结束过程，结果为'+表达式`结束函数的执行并返回结果。

例如：

```
定义过程用于求正方形的面积，定义数字类型参数表示长，定义数字类型参数表示宽，结果为数字类型。
定义数字类型值表示面积，赋值为长乘以宽。
结束过程，结果为面积。
```

编辑为Java代码为

```java
BigDecimal 求正方形面积(BigDecimal 长, BigDecimal 宽) {
    BigDecimal 面积 = 长.multiply(宽);
    return 面积;
}
```

要调用过程，使用`'执行'+过程名称+'过程，参数为'+参数列表`，参数列表中多个参数可以使用`和`连接。

如果要将执行的结果赋值给一个变量，则使用`赋值为过程的结果`语句连接变量名和执行过程。
